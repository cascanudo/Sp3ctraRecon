# Code of Conduct / Codigo de Conducta

[English](#english) | [Espanol](#español)

---

## English

We want clear, respectful, and useful collaboration for learning.

- Criticize the code, not the person.
- Explain changes clearly.
- Avoid gatekeeping.
- Be thoughtful in tone when pointing out issues.
- Prioritize evidence, context, and authorization.

This project values direct help and honest review. If something is wrong, say it with precision and respect.

---

## Español

Queremos una colaboracion clara, respetuosa y util para aprender.

- Critica el codigo, no a la persona.
- Explica cambios con claridad.
- Evita gatekeeping.
- Cuida el tono cuando senales problemas.
- Prioriza evidencia, contexto y autorizacion.

Este proyecto valora la ayuda directa y la revision honesta. Si algo esta mal, dilo con precision y con respeto.
